How to use CSIRAC emulator in this web application:

Step 1:

According to the guide from User Guide, write down the instructions in the first blank and submit the program.

Step 2:

Input the data and submit.

Step 3:

When the Monitor Status shows 'Ready', submit the frequency and start the submitted program. If you want to pause at the process of the running program, press 'Pause' and revise the PCRegister if you want.

Step 4:

Continue the progress. Read the Program Memory History, Register History and Data Memory History from the web.
